### Sobre
Estudante de **Ciência da Computação** com foco em **Análise de Dados**, atualmente atuando principalmente com **soluções SAP**.

Tenho experiência prática com diversas ferramentas e tecnologias SAP, incluindo:

- 💻 **SAP ABAP**  
- 🌐 **SAP Datasphere**
- 📊 **SAP Analytics Cloud (SAC)** 
- ☁️ **SAP CAP (Cloud Application Programming)**

Também possuo experiência com **desenvolvimento Back-end em Python** e **bancos de dados relacionais com SQL**, aplicados principalmente em projetos acadêmicos e pessoais.

Atualmente estou aprofundando meus conhecimentos em **plataformas SAP voltadas para dados e aplicações**, com o objetivo de integrar habilidades de **engenharia de dados, desenvolvimento e BI**.
